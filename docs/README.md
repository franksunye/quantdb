## 🎯 项目概览

QuantDB 是一个高性能的股票数据缓存服务，提供：

- **极致性能**: 比AKShare快98.1%，响应时间~18ms
- **国际化支持**: 专业英文界面，支持A股+港股
- **完整解决方案**: 后端API + 前端界面 + 云端部署
- **生产就绪**: 完整测试体系，稳定可靠
